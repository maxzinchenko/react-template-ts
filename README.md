# React boilerplate TS

## Packages installation

```bash
$ yarn
```

## Running the app

```bash
# development mode
$ yarn start:dev

# production mode (build is required)
$ yarn start:prod
```

## Building the app

```bash
# build to /dist folder
$ yarn build

# build to /dist folder and run the built app
$ yarn build:start
```

## Checking for the code style exceptions

```bash
# eslint check
$ yarn lint:check
```
